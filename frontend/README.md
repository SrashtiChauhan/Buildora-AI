# 🚀 Buildora AI

Buildora AI is an AI-powered website builder that generates modern, responsive websites instantly.

## ✨ Features

- ⚡ AI-powered website generation  
- 🎨 Modern UI with animations  
- 📱 Fully responsive layouts  
- 💳 Razorpay payment integration  
- 📦 Export clean production-ready code  
- 🧠 Live preview & editor  

---

## 🛠 Tech Stack

**Frontend**
- React (Vite)
- Tailwind CSS
- Framer Motion

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Payments**
- Razorpay
